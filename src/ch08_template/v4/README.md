统一一下制作流程就是：

  1. 煮开水  
  2. 用开水泡咖啡或茶  
  3. 把饮料倒进杯子里  
  4. 加调料

这就是模板方法。
模板方法定义了一个算法的步骤，并允许子类为一个或多个步骤提供具体实现。
